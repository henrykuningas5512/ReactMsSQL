# ReactMsSQL CRUD App

React + Node.js + MS SQL CRUD rakendus.

## Funktsioonid

- Kuvab mängud andmebaasist
- Lisab uue mängu
- Muudab olemasolevat mängu
- Kustutab mängu

## Tehnoloogiad

- React
- TypeScript
- Node.js
- Express
- MS SQL Server
- Axios

## Käivitamine

### Backend

```bash
cd backend
npm install
npm run dev
```
### Frontend
```bash
Frontend
cd frontend
npm install
npm run dev
```
